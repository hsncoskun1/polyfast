## 5. Onemli Ayrimlar

### Candidate Discovery vs Live-Validated Discovery
- v0.2.3: CANDIDATE discovery — aday event bulma
- v0.2.5: LIVE-VALIDATED discovery — event'in gercekten live oldugu dogrulandi
- Live validation ≠ market data/PTB/fiyat dogrulamasi

### In-Memory Authoritative State vs Persisted Support State
- Registry in-memory, authoritative karar veren katman
- Persistence (checkpoints, balance snapshots) destekleyici — domain logic uretmez
- Registry persistence Faz 7 recovery ile gelecek
