## 3. Tasarim Kararlari

- **FastAPI + aiosqlite:** Async-first backend, SQLite WAL mode
- **Pydantic validation:** Config schema with cross-field validators (min < max)
- **Migration runner:** File-based SQL migrations with version tracking
- **Credential masking:** Regex-based string masking + recursive dict masking
- **JSON logging:** Structured with entity_type/entity_id for future querying
- **Vite proxy:** Frontend proxies /api/* to backend, no CORS issues in dev
