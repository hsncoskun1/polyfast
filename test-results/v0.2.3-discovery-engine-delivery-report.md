## 3. Tasarim Kararlari

### DiscoveredEvent
- Frozen dataclass (immutable)
- Yalnizca discovery-phase modeli, registry state icermiyor
- from_api_event factory: parse basarisizsa None doner
- Asset extraction: question'dan regex-free keyword matching
- Duration extraction: 15m > 5m sirasi (substring collision onlendi)

### DiscoveryEngine
- PublicMarketClient uzerinden GET /events sorgusu
- 3 katmanli filtre: category=crypto, duration=5m, question contains up/down keywords
- Basarisiz scan → DiscoveryResult(success=False, events=[], error_message)
- Basarili scan → DiscoveryResult(success=True, events=[...])
- Connectivity failure health warning uretir, sessiz bypass YOK

### Filter Criteria
- Category: "crypto" (exact match)
- Duration: "5m" (question keyword parse)
- Up/Down: keywords ["go up", "go down", "above", "below", "higher", "lower", ...]
