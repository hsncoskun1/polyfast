## 3. Tasarim Kararlari

### Client Ayirimi

| Client | Sorumluluk | Credential | Base URL |
|--------|-----------|------------|----------|
| PublicMarketClient | Discovery, public market data | Yok | gamma-api.polymarket.com |
| AuthenticatedTradingClient | Balance, order, fill | api_key + secret + passphrase | clob.polymarket.com |
| RelayerClient | Claim (signer type 2) | relayer_key + private_key + funder | relayer.polymarket.com |

### Error Classification

| HTTP Status | Category | Retryable |
|-------------|----------|-----------|
| 401, 403 | AUTH | No |
| 429 | RATE_LIMIT | Yes |
| 400 | VALIDATION | No |
| 5xx | SERVER | Yes |
| Timeout | TIMEOUT | Yes |
| Connection | NETWORK | Yes |

### CredentialStore

- Minimum iskelet: in-memory, load/load_from_dict, credential check methods
- Trading headers builder (POLY_API_KEY, POLY_SIGNATURE, POLY_PASSPHRASE)
- Full lifecycle (persistence, masking, propagation, rebind) → Faz 3.5

### BaseClient Retry

- Configurable retry_max (default 3)
- Exponential backoff (1s, 2s, 4s)
- Only retries retryable errors (5xx, 429, timeout, network)
- Non-retryable errors (401, 400) fail immediately
