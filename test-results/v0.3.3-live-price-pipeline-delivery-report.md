## 3. Tasarim Kararlari

### Live Price != PTB
- Live price = outcomePrices (Up/Down yuzdeleri, 0.55/0.45)
- PTB = priceToBeat (event baslangic referans fiyati, $66,885)
- AYRI kavramlar, AYRI moduller, KARISTIRILMADI

### Normalization
- outcomePrices JSON string veya list olarak geliyor
- Parse: up_price (0.0-1.0), down_price (0.0-1.0)
- Spread ayrica saklanir

### Gecersiz Veri Filtreleme (CLAUDE.md kurali)
- 0 deger → INVALID (evaluation'a ULASMAZ)
- Bos string → INVALID
- Malformed JSON → INVALID
- Negatif deger → INVALID
- 1.0 ustu deger → INVALID
- None → INVALID

### Freshness
- stale_threshold_sec = 30 (default)
- Her get_record() cagrisinda freshness otomatik kontrol
- Stale → PriceStatus.STALE olarak isaretlenir
- Health incident uretilir

### PriceStatus Enum
- FRESH: gecerli ve guncel
- STALE: gecerli ama eski
- INVALID: gecersiz veri
- WAITING: henuz veri gelmedi
