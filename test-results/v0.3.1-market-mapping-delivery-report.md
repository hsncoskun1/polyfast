## 3. Tasarim Kararlari

### Mapping Modeli
- MarketMap: condition_id, asset, tokens (tuple of TokenMapping), status, mapped_at
- TokenMapping: token_id, side (UP/DOWN), outcome
- MarketSide enum: UP, DOWN
- MappingStatus enum: MAPPED, UNMAPPABLE, ERROR
- Frozen dataclass'lar

### Side Classification
- "Yes" / "Up" / "Higher" / "Above" → UP
- Diger her sey → DOWN (conservative default)
- Gercek Polymarket verisine gore refine edilecek (canli testte)

### Failure Davranisi
- Token yoksa → UNMAPPABLE (sessiz gecme YOK, log WARNING)
- condition_id yoksa → UNMAPPABLE (log WARNING)
- token_id eksik token'lar atlanir (log)
- "Muhtemelen bu market" gevsek yaklasimi YOK

### Source of Truth
- Mapping girdi: event_data dict (tokens array veya markets > tokens nested)
- condition_id → token_id → side iliskisi net
- is_complete: hem UP hem DOWN token varsa True
